## GraphQL Express Learning Repo

### Start
```bash
  # Install
    git clone https://github.com/sumit194/graphql-tutorial.git
  # Install dependencies
	npm install
  # Start server
	npm start
```

### Features

-  Static data for **Query** in **GraphQL**.
